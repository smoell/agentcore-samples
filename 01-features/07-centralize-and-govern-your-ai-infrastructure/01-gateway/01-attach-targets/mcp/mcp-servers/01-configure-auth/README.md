# Configure Authentication

## Tutorials

| Section                                             | Description                                                      |
| :-------------------------------------------------- | :--------------------------------------------------------------- |
| [client-credentials](client-credentials/)           | OAuth 2.0 client credentials (M2M) for inbound and outbound auth |
| [authorization-code-flow](authorization-code-flow/) | OAuth 2.0 authorization code flow for user-delegated auth        |
| [token-exchange](token-exchange/)                   | Token exchange for cross-domain authentication                   |
